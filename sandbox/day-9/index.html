Day 9 - Dark Mode Basics

Topic: using Tailwind's dark: variant.

Notes (What I'm Learning Today)

  • Dark mode is controlled using the dark: prefix.
  • Adding the dark class enables dark styles.
  • Only colors usually change between light and dark modes.
  • Layout and spacing typically remain the same.
