Day 4 - Responsive Breakpoints

Topic: using sm:, md:, lg: to change text, layout, and spacing at different screen sizes.

Notes (What I'm Learning Today)

  • Tailwind is mobile-first: base classes apply to all sizes, and sm:, md:, lg: override them at larger breakpoints.
  • Responsive text example: text-2xl sm:text-3xl md:text-4xl lg:text-5xl grows as the screen width increases.
  • Responsive layout example: flex flex-col md:flex-row stacks content on mobile and switches to side-by-side on medium screens.
  • Responsive spacing: p-4 md:p-8 lg:p-12 or gap-4 md:gap-8 to give content more breathing room on larger screens.
  • Responsive width: w-full md:w-1/2 lets elements occupy full width on mobile and half width on desktop.
