Day 5 - Grids & Cards

Topic: grid layouts, card components, and preparing for real project structure.

Notes (What I'm Learning Today)

  • Using grid to create two-dimensional layouts (rows + columns).
  • Defining columns with grid-cols-1, md:grid-cols-2, lg:grid-cols-3.
  • Controlling spacing between cards using gap-4, gap-6, gap-8.
  • Building reusable card styles with: bg-slate-800, p-6, rounded-xl, shadow.
  • Understanding when to use grid (sections with many items) vs flex (alignment within a single card).
