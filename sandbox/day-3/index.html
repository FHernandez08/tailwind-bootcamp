Day 3 – Typography & Colors

Topic: text sizes, font weights, text colors, and basic hierarchy on a dark background.

Notes (What I'm Learning Today)

  • How to use text-sm, text-base, text-lg, text-xl, text-2xl, etc. for font sizes.
  • How font-normal, font-medium, font-semibold, font-bold affect weight.
  • Using text colors like text-white, text-slate-100, text-slate-300, text-slate-400, and an accent like text-sky-400.
  • Line height utilities like leading-normal and leading-relaxed to improve readability.
  • Letter spacing with tracking-wide or tracking-tight (good for labels).
