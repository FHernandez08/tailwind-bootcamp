Day 6 - Reusable UI Patterns

Topic: buttons, section headers, and tags that can be reused across the site.

Notes (What I'm Learning Today)

  • Reusable UI patterns help keep styling consistent and reduce decision fatigue.
  • Buttons usually come in two main variants: primary (solid) and secondary (outline/ghost).
  • Section headers should follow the same structure and spacing across all sections.
  • Tags / pills are small UI elements used for skills, tech stacks, or categories.
  • The goal today is consistency, not perfection.
